# AI Pitch Practice

یک سیستم حرفه‌ای برای تمرین پیچ استارتاپ با استفاده از هوش مصنوعی. شامل دو فاز: ضبط و تحلیل پیچ، و سپس مصاحبه زنده با VC AI.

## ویژگی‌ها

### فاز 1: ضبط و تحلیل پیچ (Offline)
- ✅ ضبط صدا با MediaRecorder API (3-10 دقیقه)
- ✅ تبدیل صدا به متن با Whisper
- ✅ تحلیل کامل با GPT-4 (حالت VC)
- ✅ بازخورد جامع: امتیاز وضوح، نقاط قوت/ضعف، پرچم‌های قرمز
- ✅ سوالات چالشی آماده برای فاز 2

### فاز 2: مصاحبه زنده با VC AI (Real-time)
- ✅ مکالمه متنی زنده با AI در نقش VC
- ✅ سوالات چالشی بر اساس تحلیل فاز 1
- ✅ Follow-up و challenge کردن پاسخ‌های vague
- ✅ شبیه‌سازی واقعی جلسه سرمایه‌گذاری

## نصب و راه‌اندازی

### پیش‌نیازها

- Node.js 18+ 
- npm یا yarn
- API Key از OpenAI

### مراحل نصب

1. نصب وابستگی‌ها:
```bash
npm install
```

2. ایجاد فایل `.env.local`:
```bash
cp .env.example .env.local
```

3. اضافه کردن API Key:
```
OPENAI_API_KEY=your_openai_api_key_here
```

4. اجرای پروژه:
```bash
npm run dev
```

5. باز کردن مرورگر:
```
http://localhost:3000
```

## استفاده

### فاز 1: ضبط پیچ
1. روی دکمه "شروع ضبط" کلیک کنید
2. پیچ خود را به صورت کامل ارائه دهید (3-10 دقیقه)
3. پس از پایان ضبط، تحلیل خودکار شروع می‌شود
4. نتایج تحلیل شامل امتیاز، نقاط قوت/ضعف، و سوالات چالشی را مشاهده کنید

### فاز 2: مصاحبه زنده
1. پس از مشاهده نتایج فاز 1، روی "شروع فاز 2" کلیک کنید
2. AI به عنوان VC سخت‌گیر شروع به پرسیدن سوالات می‌کند
3. به سوالات پاسخ دهید (متن)
4. AI بر اساس پاسخ‌های شما follow-up می‌پرسد و challenge می‌کند

## تکنولوژی‌ها

- Next.js 14
- TypeScript
- Tailwind CSS
- OpenAI API:
  - Whisper (تبدیل صدا به متن)
  - GPT-4 Turbo (تحلیل پیچ و مصاحبه)
- MediaRecorder API (ضبط صدا)

## نکات مهم

- برای ضبط صدا، دسترسی به میکروفون لازم است
- محیط آرام برای ضبط بهتر توصیه می‌شود
- پیچ خود را به صورت کامل و بدون عجله ارائه دهید
- در فاز 2، به سوالات به صورت مستقیم و واضح پاسخ دهید
- AI می‌تواند پاسخ‌های vague را challenge کند
