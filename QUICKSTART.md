# راهنمای سریع شروع

## نصب سریع

```bash
# 1. نصب وابستگی‌ها
npm install

# 2. ایجاد فایل محیطی
echo "OPENAI_API_KEY=your_key_here" > .env.local

# 3. اجرای پروژه
npm run dev
```

## دریافت API Key از OpenAI

1. به [OpenAI Platform](https://platform.openai.com/) بروید
2. وارد حساب کاربری خود شوید
3. به بخش API Keys بروید
4. یک API Key جدید ایجاد کنید
5. آن را در فایل `.env.local` قرار دهید

## استفاده

### فاز 1: ضبط و تحلیل پیچ

1. مرورگر را باز کنید: `http://localhost:3000`
2. روی "شروع ضبط" کلیک کنید و اجازه دسترسی به میکروفون را بدهید
3. پیچ خود را به صورت کامل ارائه دهید (3-10 دقیقه)
4. روی "پایان ضبط" کلیک کنید
5. منتظر بمانید تا:
   - صدا به متن تبدیل شود (Whisper)
   - پیچ تحلیل شود (GPT-4)
6. نتایج را مشاهده کنید: امتیاز، نقاط قوت/ضعف، سوالات چالشی

### فاز 2: مصاحبه زنده با VC AI

1. پس از مشاهده نتایج فاز 1، روی "شروع فاز 2" کلیک کنید
2. AI به عنوان VC شروع به پرسیدن سوالات می‌کند
3. به سوالات پاسخ دهید (متن)
4. AI بر اساس پاسخ‌های شما follow-up می‌پرسد

## نکات

- برای ضبط بهتر، در محیط آرام باشید
- پیچ خود را به صورت کامل و بدون عجله ارائه دهید
- در فاز 2، به سوالات به صورت مستقیم و واضح پاسخ دهید
- AI می‌تواند پاسخ‌های vague را challenge کند - این طبیعی است!
